Principais formataçôes

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase, tenho um termo em destaque usando a tag <strong> (não simântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não simântico)

Nesta frase, temos um termo em Ênfanse usando a tag <em> (semântica)

Texto marcado

Nesta frase, temos um texto marcado usando a tag <mark>

Neste paráfrago temos outro texto marcado usando a tag <mark style="background-color:>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno usando a tag <big> e <small>

Não se usa mais a tag <big> pra crescer ou diminuir o texto se usa o CSS

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase usando a tag <u> (não simântica)

Podemos marcar um texto como inserido para dar uma ênfase usando a tag <ins> (simântica)

Texto sobrescrito

para inserir coisa do tipo x20+3 (x elevado a 20+3) usa a tag <sup>

Texto subescrito

para inserir coisa do tipo H2O usa a tag <sub>
